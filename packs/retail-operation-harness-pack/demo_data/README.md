# 样例数据集 (Demo Data) 说明

这是一个轻量级零售样例数据集，用于帮助用户快速体验 pi-xanthil 零售经营分析 Harness 包的能力。

## 数据集包含的表：
1. `orders.csv`：订单主表
2. `order_items.csv`：订单明细表
3. `products.csv`：商品信息表
4. `customers.csv`：客户信息表
5. `stores.csv`：门店信息表
6. `campaigns.csv`：活动信息表

## 预设业务场景问题
在使用该样例数据执行分析时，分析师 Agent 会发现以下埋设的问题点：
- 最近一个月整体销售下降。
- 老客复购率下降是核心下跌因素。
- 某核心爆品出现缺货，影响了订单数。
- 华东区域门店表现最差。

用户可以通过一键触发“销售下滑诊断”，快速体验该行业包如何定位上述问题并输出标准化诊断报告。
