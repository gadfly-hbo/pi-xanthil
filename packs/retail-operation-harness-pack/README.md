# Retail Operation Harness Pack v0.1 (零售经营分析 Harness 包)

## 简介
这是一个可安装、可运行、可评估、可交付的零售行业分析包。它不是简单的 Prompt 合集，而是一套“零售数据分析师操作系统”。第一版（v0.1）聚焦于解决最核心的问题：**销售为什么涨跌？**

## 核心能力
- 销售下滑诊断 (Sales Decline Diagnosis)
- 本地数据侦察与映射
- 自动化经营诊断与拆解
- 报告生成与质量评分器 (Eval)

## 目录结构
- `AGENTS.retail.md`: Agent 行业规则
- `data_requirements/`: 数据要求与表结构
- `ontology/`: 行业语义骨架与指标字典
- `workflows/`: 分析工作流
- `evals/`: 质量评分标准
- `report_templates/`: 报告模板
- `demo_data/`: 样例数据
- `examples/`: 输出样例
