# 零售指标字典 (v0.1 核心精简版)

## 1. 销售额 (GMV)
- **业务含义**：衡量业务规模的核心指标，通常指**实付销售额**（排除了取消和全额退款的订单）。
- **计算口径**：`SUM(orders.payment_amount)` WHERE `order_status` 为成功或部分退款。
- **注意事项**：分析时应明确是否包含退款；不能和标价金额（`order_amount`）混淆。

## 2. 订单数
- **业务含义**：发生购买行为的单次记录数，是拆解销售额的核心因子。
- **计算口径**：`COUNT(DISTINCT order_id)` WHERE `order_status` 为成功或部分退款。

## 3. 客单价 (AOV - Average Order Value)
- **业务含义**：平均每个订单的实付金额，代表用户单次购买力。
- **计算口径**：`销售额 / 订单数`。
- **注意事项**：受商品结构和连带率影响，高客单价可能因买得多，也可能因买得贵。

## 4. 购买用户数
- **业务含义**：在指定时间段内有过成功购买记录的独立用户数。
- **计算口径**：`COUNT(DISTINCT customer_id)`。

## 5. 新老客销售占比
- **业务含义**：评估增长是依赖拉新还是复购。
- **计算口径**：
  - 新客：购买时间在注册时间所在的首个业务周期内（如首月）。
  - 老客：非首次购买周期内的用户。

## 6. SKU 销售额贡献
- **业务含义**：单品的销售占比，用于识别爆品和长尾。
- **计算口径**：`SUM(order_items.item_payment)` 按 `sku_id` 分组。
