## 3. Agent-E 派发任务（T-E 系列 · 分支 feat/engine-workflow-*）

### T-E0 · 删孤儿前端 + single 退化分支〔阶段0,与 T-C0 同批〕
- 删 5 文件:`ExecutionPane` `AgentFlowPane` `FlowChatPane` `FlowWorkflowPane` `FlowEditorPane`

- 验收:-~1800 行;无悬空 import;全绿 + multi/AnaX tab 冒烟

### T-E1 · runner onBlock 回跳 + trace 迭代谱系〔阶段2,依赖 T-C3〕
- `WorkflowNode` 加 `onBlock` 字段 + `validateWorkflow` 按口径校验

- trace:每轮落 `runDir/<nodeId>/iter-<n>/`,verdict 落 `gates/<id>-iter<n>.json`
- 验收:不配 onBlock 行为零变化(向后兼容);AnaX 现有 flow 不受影响

### T-E2 · 预算/停止条件接线〔阶段2,依赖 T-C4〕
- runner 接入 T-C4 原语:`maxIterations` 上限 + run 级 token/cost 超阈中断 + 红线命中(`deterministicRedLineCheck`)硬中断**不进重试环**

### T-E3 · gate 节点编辑器 UI〔阶段2,依赖 T-C3〕

- 验收:可视化配置回跳;不破坏现有 DAG 编辑

### T-E4 · MVP SQL 修复 loop〔阶段2〕

- 验收:模板可实例化运行
