# onto-xanthil 自定义抽取 Prompt 库

> 本目录存放按业务领域定制的高阶 Prompt 模板，供 onto-xanthil「导入」tab → 「▸ 自定义抽取 Prompt」使用。
> 数据来源：服饰企业指标字典（944 条指标），按「一级业务主题」+ 关键词二次分类拆分为 9 个领域。

## 使用方式

1. 打开 onto-xanthil → 导入 tab
2. 展开「▸ 自定义抽取 Prompt」
3. 将对应领域子文件夹中的 `prompt.txt` 内容粘贴到编辑框
4. 将同文件夹中的 `metrics.xlsx` 导出为文本，粘贴到文档输入区（或直接上传 .csv/.txt）
5. 点击「开始抽取」

## 硬约束（违反会导致抽取失败）

- **`{{content}}`** 占位符必须保留，后端据此注入文档正文
- 输出 JSON 必须包含 `entities` / `relations` / `logic_rules` / `actions` 四个 key（无数据则给 `[]`）
- `relations.kind` 仅允许 `is-a` / `part-of` / `related` 三种
- 所有引用（`source`/`target`/`linkedEntities`/`linkedLogic`）必须与同批抽取的 `nameCn` 严格一致

## 分类规则说明

1. **优先使用原始数据的「一级业务主题」字段**（零售管理 / 供应链 / 商品统筹 / 仓储物流 / 会员整体 / 活跃会员 / 新会员 / 流失会员 / 人力资源管理 / 财务及预算控制）
2. **主题为空（571 行）或「其他」的行**：基于指标名称 + 指标定义的关键词匹配规则进行二次分类
3. **少数仍未命中的指标**（如达成率、目标、占比等通用概念）：归入「零售管理」作为核心经营兜底
